# 4. Use Waveshare PoE and NVMe Hat for Raspberry Pi 5

Date: 2025-11-04

## Status

Accepted

## Context

We need a way to connect the Raspberry Pi 5 to NVMe and PoE.

## Decision

We will use the Waveshare NVMe & PoE hat for Raspberry Pi 5.

It is well tested and reliable.

## Consequences

All our Raspberry Pi 5's can now,
- be powered by PoE
- access their filesystem dramatically faster.
